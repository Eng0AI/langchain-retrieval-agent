# LangChain Retrieval Agent

A LangChain.js agent with retrieval tool powered by LangGraph, Supabase vector store, and the Vercel AI SDK.

## Tech Stack

- Next.js 15
- React 18
- LangChain.js
- LangGraph
- Vercel AI SDK
- OpenAI
- Supabase (Vector Store)
- Tailwind CSS
